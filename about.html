About the Groove Writer

The Groove Writer was built to help drummers work out their grooves and turn them into printable sheet music. It is also a good tool to share a groove via email or message board because it can generate a web link that is easy to share.

How it works:

The Groove Writer is written in HTML and Javascript and runs entirely in the browser. ABC Notation is used to create the printable sheet music. When notes are changed on the screen, new ABC Notation is generated which creates the sheet music. To play the music, Groove Writer generates a MIDI file from the notes and plays them using the browser audio interface.

How it swings:

The swing slider controls the amount of "swing" that is added to the playback. It does not effect the printable sheet music. To generate swing a delay is added to the upbeat notes in the measure. The upbeat notes are the "a" and the "e" in a 16th note groove. (1e&a), and are the "&" notes in a 8th note groove. (1&2&3&4&). During MIDI generation, the program extends the duration of the downbeat notes and shortens the duration of the upbeat notes. The percentage of the swing controls the percentage of the note duration that the upbeat is delayed. (NewDelay = NormalDelay + (NormalDelay * SwingPercentage)

Author: Lou Montulli

Inspiration: I was inspired to write this app after spending lots of time trying to use ABC notation to create good looking printable sheet music. ABC is an awesome invention, but it can be very challenging to create percussion music using it.
